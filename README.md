# Customer Segmentation using Unsupervised Learning

## Description

This project is a machine learning-based customer segmentation system designed to help businesses understand customer behavior and improve decision-making. Using unsupervised learning techniques, the model groups customers into meaningful segments based on purchasing patterns.

The project uses **RFM Analysis (Recency, Frequency, Monetary)** combined with clustering algorithms such as **KMeans** to identify valuable customer groups like loyal customers, at-risk customers, growth customers, and low-engagement customers.

An interactive **Streamlit dashboard** is included to visualize customer segments, revenue contribution, and business insights.

This project is ideal for learning real-world machine learning workflows and demonstrating practical business-focused analytics.

## Installation

### Prerequisites

Make sure the following are installed:

- Python 3.9+
- pip
- Virtual Environment (recommended)

### Setup Steps

1. Clone the repository
2. Create and activate a virtual environment
3. Install required dependencies from `requirements.txt`
4. Add the dataset to the `data/raw/` folder
5. Run the preprocessing and training pipeline

---

## Usage

Run the project in the following sequence:

1. Clean raw transactional data
2. Generate RFM features
3. Train clustering model
4. Create final customer segments
5. Launch Streamlit dashboard

Use the dashboard to:

- View customer segment distribution
- Analyze revenue by segment
- Explore customer behavior
- Support targeted marketing strategies
